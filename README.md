# Chinese

## 名称
木头人播放器

## 简介
用 Electron 编写的媒体播放器，可以播放本地的音乐。
不能播放远程（网络）文件。

### 实现功能
* 可以加入文件和文件夹。
* 可以暂停、停止、上一首、下一首
* 可以顺序、乱序、循环、单曲循环

### 未来预计实现的功能
* 音量调节
* 可以播放 cue 文件
* 可以调节播放进度
* 可以最小化
* 更强大的界面操作
* 以及其他暂时没想到的
